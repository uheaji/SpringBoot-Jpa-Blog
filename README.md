## 의존성
- Spring Web
- Lombok
- Spring Data JPA
- MySQL Driver
- Spring Security
- OAuth2 Client
- Spring Web
- Spring-security-taglibs
- tomcat-jasper
- jstl


## DB 설정
```sql
create user 'cos'@'%' identified by 'cos1234';
GRANT ALL PRIVILEGES ON *.* TO 'cos'@'%';
create database cos;
```
